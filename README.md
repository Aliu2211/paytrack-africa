# PayTrack Africa

AI-powered invoice and payment tracking SaaS for Ghanaian SMEs. Multi-tenant serverless architecture on AWS (Lambda, API Gateway, DynamoDB, Cognito).

Build spec: [PayTrack_FullBuildSpec.md](PayTrack_FullBuildSpec.md).

Status: Phase 1 (core infrastructure and invoice API) in progress. Full architecture, deployment, and usage documentation lands in Phase 5.
